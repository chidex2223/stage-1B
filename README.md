## Project Description
This is a responsive Profile Card built using only HTML, CSS, and JavaScript.  
It displays user information including avatar, bio, social links, hobbies, dislikes, and live epoch time.

---

## Features
- Semantic HTML structure
- Responsive design (mobile, tablet, desktop)
- Live updating epoch time (Date.now())
- Accessible elements with alt text and focus styles
- Data-testid attributes for testing
- Social links open in new tabs safely
